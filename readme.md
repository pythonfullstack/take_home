# Take Home

### How to run the project
- Run docker compose   
    
      docker-compose up
- Run pgAdmin
    
      http://127.0.0.1:5050
      email: admin@admin.com
      password: root
- Create Server

_*Note:*_ you can check Postgres container IP address by running the following command:
    
    docker ps <container_id> | grep IPAddress

- Run the script
    
      python main.py
